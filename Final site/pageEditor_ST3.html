Feed Back Page

Technical Description

The feed.html file creates a structured feedback form for the "Feed The Future" website. It includes a header with a logo and a navigation bar linking to various pages, with the "Feedback" page marked as active for user indication. The feedback form is inside a .feedback-container div and consists of two sections: Personal Details and Feedback Details. The personal details section includes fields for name, email, phone number, and address, while the feedback section contains a text area, a rating system, and recommendation options. Additionally, a checkbox allows users to subscribe to email updates. The form provides Submit and Reset buttons for user interaction, and a footer displays copyright information and credits. The feed.css file styles the layout using Flexbox and applies background images. The navigation bar features a hover effect with a gradient transition, and the "Join" button includes a hover and scale effect. Form inputs are styled with borders, padding, and smooth transitions, while the Submit button is green and turns blue on hover, and the Reset button is red and turns light blue on hover. A blur effect is applied for a modern, glass-like UI, enhancing readability and user experience. The overall design ensures a clean, interactive, and responsive interface
