User Profile

Technical Description

The User Profile Dashboard is a structured web page designed for the "Zero Hunger" project, providing an interactive and visually appealing interface for users. The HTML file (userprofile.html) defines various sections, including a navigation bar, user details, skills, biography, goals, volunteering experience, and a contact form. It also incorporates a video background to enhance the visual experience. The CSS file (userprofile_styles.css) ensures a modern and responsive design using flexbox and grid layouts. It features a blurred glass effect (backdrop-filter) for a stylish profile card, smooth hover effects for navigation elements, and an interactive calendar layout to display volunteering availability. The "Join" button is uniquely styled with a hover effect, encouraging user engagement. The overall design prioritizes usability, aesthetics, and responsiveness, making it an effective and professional profile dashboard.

Site map

Technical Description

The sitemap webpage provides a structured visualization of a website’s navigation, enhancing user experience. The HTML file defines the page structure, linking to a CSS stylesheet for styling. It features a header with a navigation menu, a video background, and an interactive SVG-based sitemap diagram, which represents different pages as clickable nodes. The CSS styles improve aesthetics with hover effects, background gradients, and responsive design elements. This setup ensures intuitive navigation and accessibility while maintaining a visually appealing layout.
