Splash Screen

Technical Description

HTML Structure

  • meta http-equiv="refresh" tag : Redirects the user to home.html after 4 seconds, creating a splash screen effect.

  • div class="splash-screen" tag : Contains the main content of the splash screen, including the title and team members.

  • div class="vertical-text" tag : Displays the text "GROUP MEMBERS" vertically on the right side of the screen.

  • div class="team" tag : Wraps the team member cards, each containing an image and a name.

CSS Styling

  • background: linear-gradient(): Creates a gradient background for the splash screen, enhancing visual appeal.

  • animation: shimmer: Adds a shimmering effect to the background, making it dynamic.

  • @keyframes scaleDown: Animates the title to scale down from a larger size, creating an engaging entrance effect.

  • @keyframes float: Adds a floating effect to team member images, making them visually interactive.

  • flex-direction: column and flex-direction: row: Used to arrange elements vertically and horizontally, ensuring proper layout.

  • transform: translateY(-10px): Adds a hover effect to team member cards, lifting them slightly when hovered over.

Accessibility

My Strategy :

  • I focused on creating an engaging splash screen with a 4-second redirect, using animations and a gradient background for visual appeal. Accessibility was prioritized with alt text for images, high contrast for readability, and a responsive layout for various devices.

My Quality :

  • The page features clean, semantic HTML and smooth CSS animations like scaling and floating effects. Hover interactions enhance engagement, while responsive design and accessibility features ensure usability for all users, including those with disabilities.

Volunteer Page

Technical Description

HTML Structure

  • header tag : Contains the logo and navigation bar, providing easy access to other pages.

  • section tag : Used to structure the volunteer programs and reviews sections, ensuring organized content.

  • div class="card" tag : Represents each volunteer program with details like images, descriptions, and buttons.

  • table tag : Displays program details (duration, fees, age) in a structured format.

  • button: Interactive buttons for "View Project" and "Apply" to encourage user action.

CSS Styling

  • display: grid: Used for the volunteer programs section to create a responsive, two-column layout.

  • hover effects: Applied to cards and buttons (e.g., transform: translateY(-10px)) to enhance interactivity.

  • flexbox: Used in the reviews section to align and distribute review cards evenly.

  • background: linear-gradient: Adds a gradient overlay to the background for visual appeal.

  • media queries: Ensure the page is responsive, adapting to smaller screens.

Accessibility

My Strategy :

  • I focused on accessibility by adding alt text to images for screen readers, ensuring high contrast between text and background for readability, and designing keyboard-navigable interactive elements like buttons and links. This makes the page usable for everyone, including those with disabilities.

My Quality :

  • The page uses semantic HTML for clear structure, responsive design for all devices, and interactive elements with hover and focus states to enhance user experience. This ensures the page is functional, visually appealing, and accessible.

Content Page

Technical Description

HTML Structure

  • header tag : Contains the logo and page title, providing a clear introduction to the page.

  • section class="project-card" tag : Structures the main content, including an image and detailed text about the "First Harvest" project.

  • ul and li tags : Used to list key features and impacts of the project in an organized manner

  • button id="back-to-top" tag : Provides a button for users to smoothly scroll back to the top of the page.

CSS Styling

  • background: linear-gradient: Creates a visually appealing gradient background for the page.

  • project-card: A centered card with hover effects (transform: translateY(-10px)) to make it interactive.

  • flexbox: Used to align content vertically and center it on the page.

  • box-shadow: Adds depth to the card and back-to-top button for a modern look.

  • media queries: Ensures the page is responsive and adapts to different screen sizes.

Accessibility

My Strategy :

  • I focused on accessibility by adding alt text to images for screen readers, ensuring high contrast between text and background for readability, and making interactive elements like buttons and links keyboard-navigable. This ensures the page is usable for everyone, including those with disabilities.

My Quality :

  • The page uses semantic HTML for clear structure, responsive design for all devices, and interactive elements with hover and focus states to enhance user experience. This ensures the page is functional, visually appealing, and accessible.
